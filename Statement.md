Problem Statement:
  Estimating the price of a house based on its key features can be time-consuming and often requires manual research or consultation with real estate professionals. Many users need a quick and reliable way to determine approximate property values using basic inputs such as the number of bedrooms, bathrooms, total area, and location. However, there is a lack of simple tools that not only provide instant predictions but also maintain a record of past estimates for future reference.To address this issue, this project develops a machine learning–based system capable of predicting house prices from user-provided data. The system also stores each prediction in a database, enabling users to review previous results. By combining automation, data storage, and an interactive interface, the project offers an efficient and user-friendly solution for estimating housing prices.

Scope of the project :
  The scope of this project is confined to predicting house prices using a limited set of input features, namely the number of bedrooms, bathrooms, total area (in square feet), and location category. The model is trained on a predefined sample dataset and is intended for demonstration purposes rather than real-world deployment with large-scale or dynamic housing data.The system is implemented as a command-line application, where users can input property details, obtain predicted prices, and store each result in a local SQLite database. It also provides functionality to view previously saved predictions, enabling users to keep track of multiple estimates.This project primarily focuses on illustrating the integration of machine learning, database management, and user interaction within a simple application. It does not cover advanced features such as real-time data updates, inclusion of complex pricing factors, graphical user interfaces, or deployment on web or mobile platforms.

Target Users:
  This project is primarily designed for students and beginners who are interested in understanding how machine learning can be applied to solve real-world problems in an interactive manner. It is also suitable for developers and learners who want to explore the integration of predictive models with databases using Python. Additionally, the project can be useful for educators as a practical example to demonstrate key concepts such as data preprocessing, model training, and data persistence in a simple and easy-to-understand format. 

High level features: 
  The project offers an automated system for predicting house prices using a trained machine learning model. It processes user-provided property details and generates estimated prices efficiently. A data preprocessing pipeline is incorporated to convert categorical inputs like location into numerical form and to normalize numerical features, ensuring better prediction accuracy. All generated predictions, along with their corresponding input data, are stored in a SQLite database, allowing users to keep track of past results. The application features a simple menu-driven interface, enabling users to easily perform predictions, view saved records, or exit the program without requiring any technical expertise.
